# AKIO Studio

**Where Beauty Meets Code** — A digital studio at the intersection of fashion, beauty, and technology.

## Quick Start

Open `index.html` in your browser, or run a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js (npx)
npx serve
```

Then visit `http://localhost:8000`

## What's Included

- **Editorial hero** — Vogue-inspired typography and layout
- **Services** — Web apps, booking systems, event management, beauty × tech
- **Philosophy** — Your manifesto and brand story
- **Contact** — Ready for your close-up CTA

## Customization

- Update the contact email in `index.html` (search for `hello@akiostudio.com`)
- Adjust pastel colors in `styles.css` under `:root`
- Add your own imagery to the `assets/` folder
